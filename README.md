# employee-workhours-zscore-analysis
Employee Work Hours Analysis project using Python, Pandas, Seaborn, and statistical techniques like Z-score for outlier detection and distribution analysis.
Project Overview

This project analyzes employee work-hour data using Python and statistical methods. The main objective is to identify work-hour patterns, calculate statistical measures, detect outliers using Z-scores, and visualize the distribution of employee working hours.

The project demonstrates practical implementation of:

Descriptive Statistics
Standard Deviation
Z-Score Calculation
Outlier Detection
Data Cleaning
Data Visualization
Problem Statement

Organizations often need to analyze employee working-hour data to understand productivity patterns and identify unusual work behavior.

This project performs:

Statistical analysis of employee work hours
Calculation of mean and standard deviation
Detection of outliers using Z-score method
Removal of abnormal values
Visualization of normalized distribution after cleaning
Technologies Used
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook
